# 非功能需求（Non-Functional Requirements）

## 使用者體驗類

### NFR-01 易用性
描述：遊戲操作須簡單易懂
- 測試方式：初次玩家能完成餵食、清潔、互動操作
- 優先度：高

### NFR-02 教育與療癒性
描述：介面設計需具有教育和療癒效果
- 測試方式：色彩柔和、文字清楚、動畫自然、提供知識提示
- 優先度：高

### NFR-03 互動自然性
描述：互動反應需自然、符合玩家操作預期
- 測試方式：角色行為符合操作，正向反應合理，負向反應提示清楚
- 優先度：高

### NFR-06 教育提醒
描述：忽略操作或錯誤操作時需有提示
- 測試方式：玩家餵錯食物或未清潔 → 彈出警告訊息
- 優先度：高
