# 功能需求（Functional Requirements）

## User Story 1：學習飼養知識

### FR-01 建立蜜袋鼯角色
描述：系統需生成蜜袋鼯角色，包含屬性：飢餓度、清潔度、快樂度
- 輸入：遊戲開始
- 輸出：角色生成成功
- 測試方式：
  - 角色屬性值初始正確

### FR-02 餵食功能
描述：玩家能餵食蜜袋鼯，不同食物影響不同屬性
- 輸入：玩家選擇食物
- 輸出：飢餓值增加，快樂值增加，健康值變化
- 測試方式：
  - 正確餵食 → 飢餓值增加
  - 忽略餵食 → 飢餓值下降，角色顯示不開心
  - 餵錯食物 → 健康值下降，顯示警告

### FR-03 清潔籠子
描述：玩家可清理籠子
- 輸入：點擊清潔操作
- 輸出：清潔度提升，異味消失，疾病機率降低
- 測試方式：
  - 玩家清理 → 清潔度上升
  - 未清理超過 3 天 → 清潔度危險，角色出現不適

### FR-04 互動功能
描述：玩家可與蜜袋鼯互動（撫摸、遊戲）
- 輸入：玩家選擇互動方式
- 輸出：快樂值上升，角色反應
- 測試方式：
  - 每天互動 → 角色親密度提升
  - 長期不互動 → 親密度下降
